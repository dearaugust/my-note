### 环境配置

前面的章节已经说了具体怎么安装 这里就不提了

```
vue init webpack vuex-demo
```

#### 进入项目文件夹，安装依赖

```
cd vuex-demo &&  npm install
```

#### 打开项目

```
npm run dev
```

这样我们就可以在浏览器中查看了

```
http://localhost:8080/
```
